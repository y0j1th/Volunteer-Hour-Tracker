# Volunteer Hours Tracker

A simple web application that allows users to track volunteer hours for a nonprofit organization. Built using HTML, CSS, and JavaScript with LocalStorage support to persist data across sessions.

---

## 🚀 Features

- Add volunteer entries (organization, date, hours, description)
- Edit existing entries
- Delete entries
- Automatic total hours calculation
- Dark mode toggle
- Data persistence using LocalStorage (data stays after refresh)

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla JS)
- LocalStorage API

---

## 🎯 Purpose

This project was built to practice frontend web development and create a useful tool for tracking volunteer hours for nonprofit work. It also serves as a portfolio project for internship and college applications.
